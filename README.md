# Shabiha Jahan | AI & Backend Developer Portfolio

Welcome to the personal portfolio of Shabiha Jahan, an AI & Backend Developer passionate about building scalable, intelligent systems and robust APIs. This modern, visually stunning portfolio is built with React, Vite, Tailwind CSS, and Framer Motion.

## 🚀 Features
- **Dynamic & Responsive Design:** Crafted with modern aesthetics, glassmorphism, and responsive layouts to ensure it looks great on any device.
- **Interactive Animations:** Uses Framer Motion for smooth scroll animations, hover effects, and micro-interactions.
- **Working Contact Form:** Integrated with EmailJS to allow direct messages right from the portfolio.
- **Projects Showcase:** Highlights key projects including AI/ML systems, backend platforms, and full-stack applications.

## 🛠️ Tech Stack
- **Frontend:** React, TypeScript, Tailwind CSS, Framer Motion
- **Tooling:** Vite
- **Icons:** Lucide React
- **Integrations:** EmailJS for the contact form

## 💻 Running the Project Locally

To run this project on your local machine, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/sabihamishu/shabiha-portfolio.git
   cd shabiha-portfolio
   ```

2. **Install dependencies:**
   Make sure you have Node.js installed, then run:
   ```bash
   npm install
   ```

3. **Start the Development Server:**
   ```bash
   npm run dev
   ```
   The application will start, and you can view it in your browser at `http://localhost:5173/`.

## 🌐 Deployment (Vercel / Render)

This project is fully ready for deployment on modern platforms like Vercel or Render. 

1. Push your code to your GitHub repository.
2. Go to [Vercel](https://vercel.com/) or [Render](https://render.com/).
3. Create a new project and import the GitHub repository.
4. The build settings should automatically be detected:
   - **Framework Preset:** Vite (or React)
   - **Build Command:** `npm run build`
   - **Output Directory:** `dist`
5. Deploy and share your portfolio with the world!
